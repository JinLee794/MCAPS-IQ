# Customer Success Account Manager (CSAM) — Knowledge Layer Context

The CSAM is the **customer-success orchestrator**. In the knowledge layer, CSAM contributions focus on:

## What You Own

- **Weekly digests** — Maintain `knowledge/weekly/` with cross-customer summaries
- **Governance records** — Document status reviews, risk escalations, remediation plans
- **Success plan context** — Track adoption metrics, usage baselines, customer health

## Your Playbooks

- [[milestone-governance]] — Weekly governance cadence
- [[pipeline-qualification]] — Routing expansion signals back to Specialist (supporting role)

## Knowledge Layer Habits

| Cadence | Action |
|---|---|
| Weekly | Publish weekly digest, review all customer files for staleness |
| Per governance meeting | Capture status + risk in meeting note |
| Per escalation | Document risk signal and remediation path |

## Handoff Pattern

When routing expansion signals:
1. Document the growth signal in customer file
2. Create meeting note capturing the context
3. Tag relevant Specialist in the PR
4. Link back to original delivery context
